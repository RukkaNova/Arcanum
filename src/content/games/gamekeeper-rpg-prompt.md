Gamekeeper is about as minimal as an AI RPG prompt gets. Shared publicly on GitHub by a user named Tiefseetauchner, it tells the model to act as "gamekeeper," set the scene, and output a small JSON block at the top of every response — name, location, inventory — so the game state is visible and the session can be paused and picked back up later. Beyond that, it's pure freeform narration: you say what you do, the gamekeeper describes what happens.

There's no combat system, no skill checks, no world document behind it — the JSON block is a save file, not a rulebook. Left to its own devices it also leans hard on the same handful of fantasy-adventure clichés (a dark forest shows up often), since there's nothing steering it toward anything more specific. It works exactly as advertised and nothing more.

Originally written for ChatGPT, but it's plain text — it runs the same way in Gemini. A free prompt, paste it in and play. 2 out of 5.
